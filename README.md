A mod that displays a combat log window for Slay the Spire 2.

## Installation

Head to [Releases](https://github.com/davidvorona/Sts2CombatLog/releases) and download all 3 files from the latest release (`.dll`, `.json`, and `.pck`).

Once they're downloaded (and unzipped, if you downloaded the `.zip`), place them in the `Slay the Spire 2/mods` folder.
