A mod that displays a combat log window with BaseLib as a dependency.

## Installation

### Dependencies

First, you'll need the **BaseLib** mod. You can find it [here](https://github.com/Alchyr/BaseLib-StS2).

You will need to download the `.dll`, `.pck`, and `.json` from the latest release and put them in `Slay the Spire 2/mods`.

### Installing the Mod

Head to [Releases](https://github.com/davidvorona/Sts2CombatLog/releases) and download all 3 files from the latest release (`.dll`, `.json`, and `.pck`).

Once they're downloaded, place them in the `Slay the Spire 2/mods` folder.

*Both BaseLib and this mod can be in subdirectories within `mods/` if you prefer.*
